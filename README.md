🔧 Desenvolvedor de Robótica e Técnico em Mecatrônica | FGC & FTC
Sou um desenvolvedor com formação técnica em Mecatrônica, apaixonado por criar a ponte entre o hardware e o software. Minha experiência é focada no desenvolvimento de sistemas de controle para robôs de competição nos ecossistemas FIRST Global Challenge (FGC) e FIRST Tech Challenge (FTC).

Como programador na FGC Team Brasil e na equipe Techamker Challenge, aplico meus conhecimentos para desenvolver soluções robustas, desde a programação de baixo nível em C até arquiteturas de software complexas em Java e C++. Minhas principais contribuições incluem:

Arquitetura de Software com Máquinas de Estado Finitas (FSM): Projetando um código modular e confiável para gerenciar os diferentes estados e ações do robô durante a partida.

Controle de Movimento em Malha Fechada: Implementação de algoritmos de controle (como PID) com dados de encoders e giroscópio (IMU) para garantir movimentos autônomos precisos e consistentes.

Desenvolvimento de Rotinas Autônomas Dinâmicas: Criação de lógicas autônomas que reagem a condições variáveis da arena, utilizando sensores para tomada de decisão.

Integração de Visão Computacional (Limelight): Utilização de dados de visão para tarefas dinâmicas como mira automática, alinhamento com objetivos e correção de trajetória em tempo real.

🏆 Meus Projetos em Destaque
FGCLib Development Center: A Oficina Virtual para Robôs da FGC
O FGCLib Development Center é uma ferramenta web completa que criei para acelerar drasticamente o ciclo de desenvolvimento de software para equipes da FIRST Global Challenge, servindo como a interface visual oficial para a biblioteca FGCLib. O objetivo é substituir a complexidade da programação baseada em texto por um fluxo de trabalho intuitivo de "arrastar e soltar", permitindo que toda a equipe participe da criação do cérebro do robô.

Funcionalidades Principais:

Configurador Visual de Robô: Monte um "gêmeo digital" do seu robô arrastando componentes.

Mapeador de Controles Gráfico: Atribua funções do robô a botões de gamepad de forma visual.

Geração de Código Completo: Com um clique, a ferramenta gera um projeto Java completo e funcional.

Tuning em Tempo Real: Conecte-se ao robô via Wi-Fi para ajustar constantes (como ganhos de PID) ao vivo.

Este projeto foi criado para capacitar as equipes, permitindo que elas foquem mais em estratégia e engenharia e menos na complexidade do código.

Tecnologias: HTML, JavaScript, C#.
