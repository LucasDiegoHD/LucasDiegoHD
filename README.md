Sou apaixonado por criar a ponte entre o hardware e o software. Minha trajetória é focada no desenvolvimento de sistemas de controle para robôs de alta performance nos ecossistemas **FIRST Global Challenge (FGC)** e **FIRST Tech Challenge (FTC)**.

Como programador na **FGC Team Brazil** e na equipe **Techmaker Challenge**, projeto soluções que vão desde o controle de baixo nível em C até arquiteturas complexas em Java e C++, sempre com foco em otimização e eficiência.

---

## 🚀 Especialidades Técnicas

* **Arquitetura de Software (FSM):** Desenvolvimento de sistemas modulares baseados em Máquinas de Estado Finitas para gestão de comportamento do robô.
* **Controle de Movimento:** Implementação de algoritmos PID e fusão de dados de sensores (IMU/Encoders) para navegação autônoma de precisão.
* **Visão Computacional:** Integração avançada com [Limelight](https://limelightvision.io/) para alinhamento dinâmico e rastreamento de alvos em tempo real.
* **Automação para FTC:** Desenvolvimento de rotinas autônomas completas com foco em velocidade, simplicidade e otimização de código, garantindo máxima performance e confiabilidade na arena sem complexidade desnecessária.

---

## 🏆 Projetos em Destaque

### [FGCLib Studio](https://fgclib-studio.web.app/) 🚀
O **FGCLib Studio** é uma plataforma web completa que desenvolvi para revolucionar o ciclo de programação na FIRST Global Challenge. Ele atua como uma interface visual poderosa para a biblioteca **FGCLib**, permitindo que equipes foquem na estratégia e engenharia, reduzindo a barreira da sintaxe complexa.
* **Destaques:** Configuração visual de componentes, mapeamento gráfico de controles e geração automática de código Java funcional.
* **Tecnologias:** HTML, JavaScript, C#.

### [Decode 23069 (FTC Control System)](https://github.com/LucasDiegoHD/Decode_23069) 🤖
Repositório oficial com o "cérebro" do robô desenvolvido para a etapa Regional da FTC. Este projeto é a demonstração prática da minha arquitetura de código, aplicando controle em malha fechada, rotinas autônomas rápidas e toda a lógica competitiva otimizada que levamos para as arenas.
* **Tecnologias:** Java.
